# mini-inventory

Sebuah aplikasi inventory sederhana yang dibuat menggunakan bahasa pemrograman bash.

## Fitur
- Tambah barang
- Hapus barang
- Edit barang
- Cari barang
- Tampilkan semua barang

## Cara Install
1. Clone project ini
```bash
https://github.com/agungmargi/mini-inventory.git
```

## Cara Penggunaan
1. Buka terminal
2. Masuk ke direktori project
```bash
cd mini-inventory
```
3.
```bash
chmod +x inventory.sh
```
4. Jalankan script
```bash
./inventory.sh
```
